Spin the Wheel Help

This guide covers setup, spin modes, sharing, accessibility, and practical usage tips for classrooms, streams, and events.

Quick start

  1. Add entries one per line, or use Paste List / CSV import.
  2. Pick a mode: normal, elimination, weighted, or tournament.
  3. Tune spin duration, sounds, and celebration effects.
  4. Spin with the wheel, center button, or Ctrl/Cmd + K.

FAQ

Is it truly random?

Yes. Winner selection uses cryptographic browser randomness. The wheel animation then lands on the pre-selected winner.

Can I reuse wheels later?

Yes. Use Save As to keep named wheels in your browser, or export JSON for backups and transfers.

What is seed mode?

Seed mode makes outcomes deterministic and reproducible. The same seed and same sequence of spins produce the same results.

Practical tips

  • Educators: use elimination mode to draw students without repeats.
  • Streamers: enable cinematic mode for dramatic reveals.
  • Event hosts: export audit JSON after draws for verification records.
