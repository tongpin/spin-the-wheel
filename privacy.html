Privacy by default

Spin the Wheel is local-first for wheel data. Your entries, settings, and results stay in your browser unless you explicitly share or export them.

Privacy principles

  1. No accounts: you can use all features without signing up.
  2. Cloudflare Web Analytics: we use it for aggregate traffic and performance metrics.
  3. No ads or ad networks: the UI is clean and free of ad placements.
  4. Wheel data stays local: entries, settings, and results are stored in your browser.
  5. User-initiated sharing and export: wheel configurations leave your browser only when you choose to share or export them.

What data stays local

  • Wheel entries, settings, and results in browser storage.
  • Language and theme preferences in localStorage.
  • Optional saved wheels in browser storage.

Cloudflare Web Analytics

We use Cloudflare Web Analytics for aggregate traffic and performance reporting. This can include metrics such as visits, countries, referrers, device/browser categories, and page performance. We use this to understand basic site usage. We do not use it for ads, accounts, or behavioral profiling.

URL sharing

Shared links contain compressed wheel data in the URL itself. We still do not store wheel configurations on our own server, but anyone with the link can open that wheel.

Clear disclosure

This is not a no-analytics site. It is local-first for wheel data, and it also uses Cloudflare Web Analytics for aggregate reporting. If that setup changes materially, this page will be updated.
